# Code of Conduct

## Our Pledge

We are committed to providing a welcoming, friendly, and respectful environment for everyone, regardless of age, experience, gender, nationality, religion, or background.

---

## Expected Behavior

- Be respectful and kind.
- Welcome constructive feedback.
- Use inclusive language.
- Respect different viewpoints.
- Help create a positive community.

---

## Unacceptable Behavior

Examples include:

- Harassment
- Hate speech
- Personal attacks
- Discrimination
- Trolling
- Spamming
- Sharing offensive content

---

## Enforcement

Project maintainers may remove comments, issues, or pull requests that violate this Code of Conduct.

Repeated violations may result in being blocked from contributing.

---

## Scope

This Code of Conduct applies to:

- GitHub Issues
- Pull Requests
- Discussions
- Project communication

---

Thank you for helping make KrishiMitra AI a welcoming project for everyone. 🌾
