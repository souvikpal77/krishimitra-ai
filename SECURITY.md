# Security Policy

## Supported Versions

The latest version of KrishiMitra AI is actively maintained and receives security updates.

| Version | Supported |
|---------|-----------|
| Latest | ✅ Yes |
| Older Versions | ❌ No |

---

## Reporting a Security Vulnerability

If you discover a security vulnerability, please do not create a public GitHub issue.

Instead, report it responsibly by contacting:

**Email:** souvikpal.dev@gmail.com

Please include:

- Description of the vulnerability
- Steps to reproduce
- Screenshots (if applicable)
- Suggested fix (optional)

We will review your report as soon as possible and work to resolve confirmed security issues.

---

## Security Best Practices

KrishiMitra AI aims to:

- Protect user privacy.
- Never intentionally collect unnecessary personal data.
- Follow secure coding practices.
- Keep dependencies updated whenever possible.
