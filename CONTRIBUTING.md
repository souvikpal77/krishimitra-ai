# Contributing to KrishiMitra AI

Thank you for your interest in contributing to KrishiMitra AI! 🌾

We welcome contributions that improve the project, fix bugs, enhance documentation, or introduce useful new features.

## How to Contribute

1. Fork this repository.
2. Create a new feature branch.

```bash
git checkout -b feature/your-feature-name
```

3. Make your changes.
4. Test your changes before committing.
5. Commit using a clear message.

```bash
git commit -m "Add: your feature"
```

6. Push your branch.

```bash
git push origin feature/your-feature-name
```

7. Open a Pull Request.

---

## Contribution Guidelines

- Follow clean coding practices.
- Write meaningful commit messages.
- Keep pull requests focused on a single feature or bug fix.
- Update documentation if required.
- Test your changes before submitting.

---

## Reporting Bugs

Please include:

- Operating System
- Browser
- Steps to reproduce
- Expected behavior
- Actual behavior
- Screenshots (if applicable)

---

## Feature Requests

Feature requests are welcome.

Please describe:

- The problem
- Your proposed solution
- Why it improves the project
